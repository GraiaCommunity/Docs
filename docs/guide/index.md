# 引言

众所周知，学习代码最好的办法，就是举例子。

所以我们将创建**大家最喜欢的涩图机器人**为例子，将你们带入机器人的世界。

本文档的所有例子都可以在 :point_right: [这里](https://github.com/GraiaCommunity/EroEroBot) 找到。

假设文档有一些专有名词不了解，也可以看看[这里](/appendix/terms)。

::: danger 警告一下
这个文档只是社区的一小部分，不是社区的所有。
整个社区真的不全是涩涩人，真的！
:::
