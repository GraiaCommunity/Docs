## 撤回消息
为了确保**涩图**能够传承下去，我们需要懂得撤回涩图  
那涩图怎么撤回呢？怎么才能知道你要撤回的是哪一条消息呢？  
第一个问题其实挺好解决的  
假设你真的会无聊翻阅 `Ariadne` 的 `docstring` 的话，你应该很快就会知道撤回的方法是
```python
app.recallMessage()
```
但问题是，该怎么样才能他，你要撤回的消息是什么呢？
